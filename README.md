# IT Graduate Portfolio

Modern personal portfolio built with React (Vite), Tailwind CSS, and Framer Motion. Includes dark/light mode, glassmorphism styling, and responsive sections for a complete IT graduate profile.

## Tech Stack

- React (Vite)
- Tailwind CSS
- Framer Motion

## Getting Started

1. Install dependencies:
	npm install
2. Start the dev server:
	npm run dev
3. Build for production:
	npm run build
4. Preview the production build:
	npm run preview

## Customize Content

- Update profile details, section copy, and project data in src/data/portfolio.js
- Tweak layout or UI structure in src/sections and src/components
- Add your resume file at public/resume.pdf to enable the download button

## Notes

- Theme toggling is handled in src/app/App.jsx using a Tailwind dark class on the html element
- Global styling and UI utilities live in src/styles/globals.css
